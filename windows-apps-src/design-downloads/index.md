# <a name="design-toolkits-and-resources-for-uwp-apps"></a>Projetar kits de ferramentas e recursos para aplicativos UWP
<link rel="stylesheet" href="https://az835927.vo.msecnd.net/sites/uwp/Resources/css/custom.css"> 

Esta seção contém design e downloads relacionados à interface do usuário para aplicativos UWP. Para ferramentas adicionais, como o Visual Studio, consulte [nossa página principal de downloads](https://developer.microsoft.com/downloads). 


## <a name="design-toolkits"></a>Kits de ferramentas de design

Estes kits de ferramentas fornecem controles e modelos de layout para o design de aplicativos UWP.

* [Kit de ferramentas do Adobe XD](http://download.microsoft.com/download/1/3/C/13CE5C92-024E-40C8-A867-83D891CC4D39/Adobe XD design toolkit for UWP apps v1707.zip)
* [Kit de ferramentas do Adobe Illustrator](http://download.microsoft.com/download/C/C/4/CC478BD5-6469-450D-80EF-52D27EEAD4C4/Adobe Illustrator design toolkit for UWP apps v1705.zip)
* [Kit de ferramentas do Adobe Photoshop](http://download.microsoft.com/download/C/D/A/CDA7FC2B-D5EA-48A4-906D-C6F4A4B311BB/Adobe Photoshop design toolkit for UWP apps v1705.zip)
* [Kit de ferramentas do Framer (no GitHub)](https://github.com/Microsoft/windows-framer-toolkit)
* [Kit de ferramentas do Sketch](http://download.microsoft.com/download/B/4/C/B4C6B0B9-C8BC-4CE2-A833-DCE89399060E/Sketch design toolkit for UWP apps v1707.zip)
