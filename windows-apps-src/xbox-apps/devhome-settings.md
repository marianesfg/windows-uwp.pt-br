# <a name="settings-page-dev-home"></a>Página de configurações (Dev Home)
   
  
A página de **configurações** da página inicial do Dev fornece acesso a várias configurações para o console de desenvolvimento.   
 ![Página de configurações de Home de desenv.](images/devhome_settings.png)   
  
A página de **configurações** contém os seguintes painéis:   
 
   *  [Configurações do console](#ID4EEB)  
   *  [Configurações de atualização de recuperação do sistema operacional](#ID4EOB)  
   *  [Visual Studio](#ID4EYB)  
   *  [Configurações de depuração](#ID4ECC)  
   *  [Configuração do script autônoma](#ID4EMC)  
   *  [Experiência inicial padrão](#ID4E3C)  

 
<a id="ID4EEB"></a>

   

## <a name="console-settings"></a>Configurações do console  
   
  
O painel de **Console configurações** fornece uma exibição "em um relance" de informações e configurações do console. Você pode usar essas ferramentas para exibir e editar o nome de host do console, área restrita e configurações de chave de acesso de segurança.   
  
<a id="ID4EOB"></a>

   

## <a name="os-recovery-update-settings"></a>Configurações de atualização de recuperação do sistema operacional  
   
  
O painel de **recuperação do sistema operacional atualizar configurações** configura definições de atualização automática do console. Observe que as alterações nessas configurações pode demorar até 24 horas entre em vigor.   
  
<a id="ID4EYB"></a>

   

## <a name="visual-studio"></a>Visual Studio  
   
  
O painel do **Visual Studio** é usado para emparelhar o console com o Visual Studio. Você também pode remover todas as instâncias do Visual Studio emparelhadas.   
  
<a id="ID4ECC"></a>

   

## <a name="debug-settings"></a>Configurações de depuração  
   
  
O painel de **configurações de depuração** é usado para exibir e alternar comuns configurações de depuração.   
  
<a id="ID4EMC"></a>

   

## <a name="unattended-script-configuration"></a>Configuração do script autônoma  
   
  
O painel de **configuração de script Unattended** gerencia os scripts de instalação para um Xbox consoles.   
  
<a id="ID4E3C"></a>

   

## <a name="default-home-experience"></a>Experiência inicial padrão  
   
  
O painel **Padrão experiência inicial** determina se a tela inicial de varejo ou a tela [Inicial do desenvolvedor no Console (Dev Home)](dev-home.md) é exibida quando o console é iniciado.   
  
<a id="ID4EJD"></a>
