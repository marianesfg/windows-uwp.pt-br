# <a name="networking-page-dev-home"></a>Página de rede (Dev Home)
   
  
A página de **rede** da casa Dev simula várias condições de redes para fins de solução de problemas. Ele também permite que você inicie um rastreamento de pacotes de rede.   
 ![Página de rede do Dev Home](images/devhome_networking.png)   
  
A página de **rede** contém os seguintes painéis:   
 
   *  [Simulação de rede](#ID4EEB)  
   *  [XBTrace (netcap)](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="network-simulation"></a>Simulação de rede  
   
  
Painel de **simulação de rede** executa uma simulação de rede a ser testado em relação a vários perfis de largura de banda.   
  
<a id="ID4EOB"></a>

   

## <a name="xbtrace-netcap"></a>XBTrace (netcap)  
   
  
O painel de **XBTrace (netcap)** inicia ou para o rastreamento de pacotes de rede.
