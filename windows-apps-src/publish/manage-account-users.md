# <a name="manage-account-users"></a>Gerenciar usuários de contas

Você pode usar o Azure Active Directory para adicionar e gerenciar usuários adicionais no seu [Partner Center](https://partner.microsoft.com/dashboard) conta. Você pode definir a função ou as permissões personalizadas que cada usuário deve ter. Você também pode atribuir uma função a um grupo de usuários, ou a um aplicativo Azure AD.

Para adicionar e gerenciar usuários de conta, você deve primeiro associar sua conta do Centro de parceiro do Active Directory do Azure da sua organização. 

Esta seção descreve como fazer o seguinte:

-   [Associar o Active Directory do Azure com sua conta no Partner Center](associate-azure-ad-with-dev-center.md)
-   [Adicionar usuários, grupos e aplicativos do Azure AD para sua conta no Partner Center](add-users-groups-and-azure-ad-applications.md)
-   [Definir funções e permissões personalizadas para usuários de conta](set-custom-permissions-for-account-users.md)

> [!TIP]
> Esses tópicos são específicos para o programa de desenvolvedores de aplicativos do Windows no Partner Center, mas associando um locatário e gerenciando usuários funcionam da mesma forma para contas no programa de desenvolvimento de Hardware do Windows (consulte [painel administração](https://docs.microsoft.com/windows-hardware/drivers/dashboard/dashboard-administration) Para obter mais informações) ou no programa de aplicativo da área de trabalho do Windows (consulte [programa de aplicativo de área de trabalho do Windows](https://docs.microsoft.com/windows/desktop/appxpkg/windows-desktop-application-program#add-and-manage-account-users) para obter mais informações).
