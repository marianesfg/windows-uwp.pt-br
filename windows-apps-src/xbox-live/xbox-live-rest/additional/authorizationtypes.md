# <a name="authorization-types"></a>Tipos de autorização
 
Serviços do Xbox Live usa um número de tipos de autorização com solicitações. Este tópico descreve os tipos de autorização padrão.
 
| Tipo| Descrição| 
| --- | --- | 
| Usuário | O chamador é um usuário autorizado do Xbox Live. | 
| Título | O chamador é um título em uma plataforma confiável.|
