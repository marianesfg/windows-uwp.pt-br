# <a name="batch"></a>/batch
Acessa as solicitações em lote complexos para várias estatísticas do player em vários títulos. O domínio para esses URIs é `userstats.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[POST](uri-batchpost.md)

&nbsp;&nbsp;Método que funciona como um método GET para solicitações em lote complexos para várias estatísticas do player em vários títulos de POST.
