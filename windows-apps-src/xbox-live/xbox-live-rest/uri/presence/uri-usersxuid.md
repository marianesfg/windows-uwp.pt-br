# <a name="usersxuidxuid"></a>/users/xuid({xuid})
Acesso a presença de outro usuário ou cliente. O domínio para esses URIs é `userpresence.xboxlive.com`.
 
  * [Parâmetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parâmetros do URI
 
| Parâmetro| Tipo| Descrição| 
| --- | --- | --- | 
| xuid| inteiro sem sinal de 64 bits| Xbox usuário ID (XUID) do usuário de destino.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[GET (/users/xuid({xuid}))](uri-usersxuidget.md)

&nbsp;&nbsp;Descubra a presença de outro usuário ou cliente.
