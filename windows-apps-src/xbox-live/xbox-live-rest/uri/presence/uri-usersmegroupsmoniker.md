# <a name="usersmegroupsmoniker"></a>/users/me/groups/{moniker}
Acessa o PresenceRecord para o meu grupo. O domínio para esses URIs é `userpresence.xboxlive.com`.
 
  * [Parâmetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parâmetros do URI
 
| Parâmetro| Tipo| Descrição| 
| --- | --- | --- | 
| moniker| cadeia de caracteres| Cadeia de caracteres define o grupo de usuários. O moniker aceitação somente no momento é "Pessoas", com uma letra maiuscula 'P'.| 
  
<a id="ID4ERB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[OBTER (/ usuários/me/grupos / {moniker})](uri-usersmegroupsmonikerget.md)

&nbsp;&nbsp;Obtém o PresenceRecord para meu grupo.
