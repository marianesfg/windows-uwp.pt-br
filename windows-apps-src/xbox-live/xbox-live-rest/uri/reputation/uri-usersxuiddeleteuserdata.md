# <a name="usersxuidxuiddeleteuserdata"></a>/users/xuid({xuid})/deleteuserdata
Redefine completamente os dados de reputação de um usuário de teste. Somente para teste. O domínio para esses URIs é `reputation.xboxlive.com`. Esse URI é sempre chamado na porta 10443.
 
  * [Parâmetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parâmetros do URI
 
| Parâmetro| Tipo| Descrição| 
| --- | --- | --- | 
| xuid| inteiro sem sinal de 64 bits| Xbox usuário ID (XUID) do usuário cujos dados estão sendo excluídos.|
