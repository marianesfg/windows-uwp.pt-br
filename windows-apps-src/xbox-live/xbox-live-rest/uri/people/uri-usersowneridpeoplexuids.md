# <a name="usersowneridpeoplexuids"></a>/users/{ownerId}/people/xuids
Pessoas de acessos por XUID da coleção de pessoas do chamador. O domínio para esses URIs é `social.xboxlive.com`.
 
  * [Parâmetros de URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Parâmetros do URI
 
| Parâmetro| Tipo| Descrição| 
| --- | --- | --- | 
| ownerId| cadeia de caracteres| Identificador do usuário cujo recurso está sendo acessado. Deve corresponder ao usuário autenticado. Os valores possíveis são "me", xuid({xuid}) ou gt({gamertag}).| 
  
<a id="ID4EOB"></a>

 
## <a name="valid-methods"></a>Métodos válidos

[POST](uri-usersowneridpeoplexuidspost.md)

&nbsp;&nbsp;Obtém as pessoas por XUID de pessoas do chamador de coleção.
