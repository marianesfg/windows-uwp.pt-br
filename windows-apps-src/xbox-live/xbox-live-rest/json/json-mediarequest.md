# <a name="mediarequest-json"></a>MediaRequest (JSON)
 
<a id="ID4EO"></a>

 
## <a name="mediarequest"></a>MediaRequest
 
O objeto MediaRequest tem a seguinte especificação.
 
| Membro| Tipo| Descrição| 
| --- | --- | --- | 
| id| cadeia de caracteres| Identificador da mídia usado pelo catálogo do Bing ou o catálogo de provedor.| 
| idType| cadeia de caracteres| Como interpretar <b>id</b>. Os valores possíveis incluem "bing" e "provedor".|
