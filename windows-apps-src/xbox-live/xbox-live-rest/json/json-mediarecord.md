# <a name="mediarecord-json"></a>MediaRecord (JSON)
 
<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
O objeto MediaRecord tem a seguinte especificação.
 
| Membro| Tipo| Descrição| 
| --- | --- | --- | 
| id| cadeia de caracteres| Identificador da mídia usado pelo catálogo do Bing ou o catálogo de provedor.| 
| idType| cadeia de caracteres| Como interpretar o identificador de mídia. Os valores possíveis incluem "bing" e "provedor".| 
| name| cadeia de caracteres| Nome localizado do conteúdo de mídia.|
