# <a name="queryclipsresponse-json"></a>QueryClipsResponse (JSON)
Encapsula a lista de clipes de jogos retornados juntamente com informações de paginação para a lista. 
<a id="ID4EN"></a>

 
## <a name="queryclipsresponse"></a>QueryClipsResponse
 
O objeto QueryClipsResponse tem a seguinte especificação.
 
| Membro| Tipo| Descrição| 
| --- | --- | --- | 
| <b>gameClips</b>| matriz de clipe de jogo| Uma matriz de clipes de jogos que atendido a consulta até o limite de solicitação (<b>maxItems</b>).| 
| <b>pagingInfo</b>| PagingInfo| Contém as informações necessárias para continuação e de paginação para chamadas subsequentes para as listas que exceder o limite de solicitação (<b>maxItems</b>).| 
  
<a id="ID4E2B"></a>
