# <a name="tools-for-profiling-and-performance"></a>Ferramentas de criação de perfil e de desempenho


A Microsoft oferece várias ferramentas para ajudar a melhorar o desempenho do aplicativo da Plataforma Universal do Windows (UWP). Siga estes links para saber como usar essas ferramentas.

**Ferramenta de resposta da IU de XAML no Visual Studio**. Uma das melhores ferramentas a ser usada para medir o impacto no desempenho dentro de seu aplicativo é a ferramenta de capacidade de resposta XAML da interface do usuário. Isso foi [atualizado](http://blogs.msdn.com/b/wpf/archive/2015/01/14/new-ui-performance-analysis-tool-for-wpf-applications.aspx) para suportar cenários ainda mais.

Consulte também:
