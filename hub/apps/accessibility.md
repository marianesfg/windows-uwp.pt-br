# <a name="accessibility-in-windows-10"></a>Acessibilidade no Windows 10

![Hero-Accessibility-bar-Smaller. png](images/hero-accessibility-bar-smaller.png)

## <a name="build-accessibility-into-your-applications-to-empower-people-of-all-abilities"></a>Crie acessibilidade em seus aplicativos para capacitar as pessoas de todas as habilidades

Produtos e serviços — incluindo mídia eletrônica — são acessíveis quando são projetados para fornecer experiências completas e bem-sucedidas para o máximo de pessoas possível.

Crie aplicativos do Windows acessíveis e inclusivos, com funcionalidade e usabilidade aprimoradas, para pessoas com deficiências (temporárias e permanentes), preferências pessoais, estilos de trabalho específicos ou restrições de situação (como espaços de trabalho compartilhados, condução, culinária, anti-reflexo e assim por diante). Algumas soluções comuns incluem o fornecimento de informações em formatos alternativos (como legendas em um vídeo) ou a habilitação do uso de tecnologias assistenciais (como leitores de tela).

**Todos devem ter acesso às mesmas salas em um edifício, quer eles precisem usar o escadas ou o elevador.**

Esta página fornece informações sobre como as várias estruturas de desenvolvimento do Windows fornecem suporte de acessibilidade para desenvolvedores que criam aplicativos para Windows, desenvolvedores de tecnologia assistencial criando ferramentas como leitores de tela e ampliadores e software engenheiros de teste Criando scripts automatizados para testar aplicativos.
